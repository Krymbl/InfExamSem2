Задание:

Есть файл "StudentsHW.txt", в котором хранятся данные о студентах в виде "Имя, Средний балл, группа" (Danil, 60, 11-403).
Нужно считать его.

Создать класс Student, у которого будут String name(имя), Integer grade(ср. балл), String group(группа).

1 задание. Найти всех студентов 11-407 группы, записать в файл "Students11-407.txt" их в виде "Имя, Средний балл, группа"
(Katya, 60, 11-407).

2 задание. Найти всех студентов, у которых средний балл больше 89, записать в файл "Students_grade89+.txt" их в виде
"Имя, Средний балл, группа" (Darya, 99, 11-402).

3 задание. Найти всех студентов 11-403 группы, у которых средний балл больше 60 и меньше 90. Записать в файл
"TheBestStudents.txt" их в виде "Имя, Средний балл, группа" (Bogdan, 80, 11-403).

4 задание. Записать в файл "Groups.txt" все группы студентов из "Students.txt" (уникальные значения групп).

5 задание. Создайте список всех встречаемых имен. Использовать только stream API,
используйте stream.map. В этом задании я подразумеваю, что вы студентов храните в списке(можете в массиве, но не надо так :) ),
превращаете список в поток и с помощью конвейерной операции map измените Студента на его имя и сохраните в списке имен

Условия: выполняйте задания исключительно через stream API
